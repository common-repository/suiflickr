== Description ==

Easy to add your flickr photos to your post while you are adding a post

== Installation ==

1. Extract all the contents of the plugin archive into your `wp-content/plugins/suiFlickr` directory.
1. Go to your wordpress admin pages and click on `Plugins` and activate the plugin called `suiFlickr`.
1. Go to your wordpress admin and click on `Options` then `suiFlickr`.
1. Set your configuration option values and click `save change` to save.
